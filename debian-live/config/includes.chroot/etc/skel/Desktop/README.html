Testing Compilers

You can test all compilers by opening a terminal from the Applications / Terminal Emulator menu, changing directories with cd ~/Desktop/test-compilers, and running make. make will compile and run example programs in each language with matching options to Kattis.

Languages Supported

  • C (using gcc 11.3.0)
  • C++ (using g++ 11.3.0)
  • Java (using OpenJDK 17.0.13)
  • Python 3 (using PyPy 7.3.11 for Python 3.9.16)

IDEs / Text Editors Provided

All IDEs and text editors are provided AS IS, with no PRE-CONFIGURATION. It is up to you to know how to configure and use your preferred software.

  • Eclipse 2024-09 (for C/C++, Java, and Python)
  • emacs
  • nano
  • PyCharm Community Edition 2024.2.4
  • vim
  • Visual Studio Code

How to Compile and Run Your Code from the Command Line

All compiler and run options below are the same options that Kattis uses. Open a terminal from the Applications / Terminal Emulator menu, cd to wherever your source code is saved, then run the command shown below for your preferred language.

C

Compile

gcc -x c -g -O2 -std=gnu11 -static -o program program.c -lm

Build

./program

C++

Compile

g++ -x c++ -g -O2 -std=gnu++17 -static -o program program.cpp

Build

./program

Java

Compile

javac -encoding UTF-8 -sourcepath . -d . program.java

Build

java -Dfile.encoding=UTF-8 -XX:+UseSerialGC -Xss64m -Xms{memlim}m -Xmx{memlim}m -cp .:./* program where {memlim} is the designated memory limit for the particular problem.

Python 3

pypy3 program.py
